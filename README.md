# Dashboard-Analytics
Interactive dashboard that analyzes global salary trends in data, allowing professionals and recruiters to explore averages, job titles,
and work arrangements through customized filters.

Objective
To analyze the evolution and distribution of salaries across the data landscape, focusing on Data Science, Engineering, and Analytics.
Tech Stack
Language: Python
Data Manipulation: Pandas
Visualization: Plotly
Web Framework: Streamlit
Key Features
Automated KPIs: Instant calculation of average salary, maximum compensation, and record frequency.
Top 10 Rankings: Visual identification of the highest-paying job titles.
Statistical Distribution: Insights into salary ranges and density.
Global Heatmap: Salary distribution mapped by professional residency.
Work Model Analysis: Comparative view of Remote vs. In-person proportions.
Project developed during Alura's "Imersão Dados com Python" event. (Projeto desenvolvido durante a Imersão Dados com Python da Alura)
